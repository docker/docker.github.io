You can check the Docker Trusted Registry (DTR) logs from a DTR node.

From a DTR node, issue the command docker logs <container-id> for the DTR containers associated with these images:

  • docker/dtr-nginx
  • docker/dtr-api
  • docker/dtr-registry
  • docker/dtr-rethink
  • docker/dtr-etcd
