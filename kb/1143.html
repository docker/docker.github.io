--- title: "How do I disable automated build of a repository on Docker Cloud?" id: 1143 draftstate: inactive deleted: false source: https://success.docker.com/@api/deki/pages/1143/contents tags: - tag: "article:topic" - tag: "product:cloud" - tag: "stage:reviewed" --- {% raw %}

Docker Cloud automated builds for a repository can be disabled using the following steps:

  1. Navigate to the repository that you wish to disable automated builds for.
  2. Click Edit to change the repository settings.
  3. Click the delete button (the trash can icon) under Tag mappings to remove the tag link for Autobuild.
