Question

How can I specify the GitHub account I would like to link to in Docker Cloud?

Answer

Documentation for linking to GitHub in Docker Cloud can be found here: https://docs.docker.com/docker-cloud/builds/link-source/

Note: If you're wanting to unlink your current GitHub account and relink to a new GitHub account, make sure to completely logout of GitHub ( https://github.com/ ) before linking via Docker Cloud.
