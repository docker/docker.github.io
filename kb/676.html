Question

  • How do I download Docker products?
  • How do I download Docker Trusted Registry (DTR)?
  • Where do I download Docker CS Engine?

Answer

For Docker CS Engine

RPM and DEB packages are available at http://packages.docker.com/ . For detailed instructions on downloading and installing Docker CS Engine please see the Installation Guide .

For Docker Trusted Registry (DTR)

  1. Docker Trusted Registry is available as a Docker image hosted on Docker Hub . For detailed instructions on downloading and installing Docker Trusted Registry (DTR) please see the Installation Guide .
